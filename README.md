# 简介
便于记录小学生习题，并可随机选题的简单windows程序。
# 环境要求
1. [.NetFramework4.6.2](https://dotnet.microsoft.com/download/dotnet-framework)
2. 开发需要[DevExpress17.2](https://www.devexpress.com)的支持
